# Day 2 – Prompt Engineering Challenge

## Objective
Today I explored the difference between a basic prompt (Lazy Prompt) and a well-structured Engineered Prompt using Claude AI.

## Tasks Completed
- Created an image using a simple (Lazy) prompt.
- Generated the same output using an Engineered Prompt.
- Compared both outputs based on:
  - Explanation quality
  - Response structure
  - Examples
  - Image design
- Documented my observations.
- Uploaded the work to GitHub.

## Lazy Prompt
Create an image explaining Prompt Engineering.

## Engineered Prompt
Create a visually appealing infographic that explains Prompt Engineering for beginners. Include:
- Definition
- Why Prompt Engineering is important
- Types of prompts
- Best Practices
- Real-world examples
- Modern color palette with icons
- Clear headings and sections
- Easy-to-read layout

## Key Learnings
- A detailed prompt produces significantly better results.
- Clear instructions improve AI response quality.
- Structured prompts generate more organized and visually appealing outputs.
- Prompt Engineering helps communicate requirements effectively to AI.

## Comparison

### Lazy Prompt Output
- Basic explanation
- Limited details
- Simple layout
- Less informative

### Engineered Prompt Output
- Well-structured response
- Better visual design
- More examples
- Professional presentation
- Easy to understand

## Conclusion
Today's activity demonstrated that the quality of AI-generated content depends greatly on the quality of the prompt. Prompt Engineering is an essential skill for obtaining accurate, detailed, and high-quality AI responses.
