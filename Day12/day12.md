# Day 12 — Job Search & Personal Branding Toolkit

## Objective
Use Claude to convert my resume into a complete, ready-to-send job search toolkit — cover letter, outreach emails, LinkedIn messaging, skill gap analysis, and interview prep — tailored for entry-level Data Analytics / Software Development roles.

## Setup
- Claude effort level: Low
- Input: latest resume (PDF)
- Target role: Data Analyst / Entry-Level Software Developer
- Prompt used: Job Search & Personal Branding Toolkit prompt (12-section structure)

## What Claude Generated
1. ATS-friendly cover letter (350–450 words)
2. Recruiter outreach email
3. Hiring manager email
4. LinkedIn connection request (under 300 characters)
5. Referral request message
6. 5-day follow-up email
7. 30-second professional introduction
8. Top 10 job titles matched to my profile
9. Key strengths recruiters will notice
10. Skill gap analysis (current strengths, missing skills, learning roadmap)
11. Personal brand summary (UVP, positioning statement, recruiter headline)
12. Interview talking points (achievements, stories, likely questions)

## Key Learnings
- The skill gap analysis was the most valuable output — it identified advanced SQL, a BI tool (Power BI/Tableau), and a Pandas/NumPy project as the biggest gaps versus typical entry-level JD requirements, with a week-by-week roadmap to close them.
- Having every message type (cover letter, recruiter DM, referral ask, follow-up) generated in one pass keeps tone and positioning consistent across every touchpoint.
- The recruiter headline and positioning statement are now going straight onto my LinkedIn profile.

## Next Steps
- Run the second improvement prompt against ATS recommendations
- Update LinkedIn headline and summary using Section 11 output
- Start closing the SQL/Power BI gap flagged in Section 10
